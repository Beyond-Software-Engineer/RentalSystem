## 1. 系统概述

欢迎使用租房系统！本系统旨在帮助您快速找到理想的租房房源，提供便捷的房源搜索、筛选和详情查看功能。

**主要功能**：
- 手机号验证码快速登录
- 房源列表浏览与搜索
- 多维度筛选（区域、价格、房型）
- 房源详情查看
- 个人中心管理

## 3. 登录注册

### 3.1 登录流程

1. 打开首页，点击底部导航"我的"
2. 点击"登录"按钮进入登录页
3. 输入手机号，点击"发送验证码"
4. 输入收到的6位验证码
5. 勾选用户协议，点击"登录"

### 3.2 注意事项

- ✅ 验证码有效期为 **5分钟**
- ✅ 发送间隔为 **60秒**
- ✅ 仅支持 **中国大陆手机号**（11位）
- ✅ 登录状态持续 **2小时**，过期后需重新登录

### 3.3 退出登录

在个人中心页面点击"退出"按钮即可退出登录。

## 5. 房源搜索

### 5.1 搜索功能

1. 在搜索框输入关键词
2. 点击搜索按钮或按回车键
3. 支持搜索房源标题、地址等信息

### 5.2 筛选条件

**区域筛选**：
- 支持按城市区域筛选
- 区域列表根据数据库动态加载

**价格筛选**：
| 选项 | 价格范围 |
|------|----------|
| 全部 | 不限制 |
| 2000以下 | 租金 < 2000 |
| 2000-3000 | 2000 ≤ 租金 < 3000 |
| 3000-5000 | 3000 ≤ 租金 < 5000 |
| 5000以上 | 租金 ≥ 5000 |

**房型筛选**：
| 选项 | 说明 |
|------|------|
| 全部 | 不限制 |
| 一居室 | 1室 |
| 两居室 | 2室 |
| 三居室 | 3室 |
| 四居及以上 | 4室及以上 |

### 5.3 排序方式

| 排序方式 | 说明 |
|----------|------|
| 最新发布 | 按发布时间降序 |
| 价格最低 | 按租金升序 |
| 价格最高 | 按租金降序 |
| 面积最大 | 按面积降序 |

## 8. 常见问题

### 8.1 登录相关

**Q：收不到验证码怎么办？**

A：请检查以下几点：
1. 确认手机号输入正确
2. 检查手机是否欠费或信号不好
3. 等待60秒后重新发送
4. 查看短信是否被拦截

**Q：验证码过期了怎么办？**

A：验证码有效期为5分钟，过期后请点击"重新发送"获取新验证码。

### 8.2 房源相关

**Q：房源图片无法显示怎么办？**

A：请检查网络连接，刷新页面重试。如果仍然无法显示，请联系客服。

**Q：如何联系房东？**

A：在房源详情页面点击"联系房东"按钮（功能开发中）。

### 8.3 其他问题

**Q：忘记密码怎么办？**

A：本系统采用手机号验证码登录，无需密码，直接使用验证码登录即可。

**Q：登录状态会保持多久？**

A：登录状态持续2小时，过期后需要重新登录。
